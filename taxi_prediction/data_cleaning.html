Section 8: Imputing Missing Values for Quantitative Variables¶

Strategy for filling NAs in quantitative variables:

  • 'Base_Fare': Use mean (numerical and continuous)

  • 'Passengers_Count': Use mode (numerical and discrete)

  • 'Per_Minute_Rate': Calculate from other variables using the formula:

    (Trip_Price - Base_Fare - Per_Km_Rate * Trip_Distance_km) / Trip_Duration_Minutes

  • 'Per_Km_Rate': Calculate from other variables using the formula:

    (Trip_Price - Base_Fare - Per_Minute_Rate * Trip_Duration_Minutes) / Trip_Distance_km

  • 'Trip_Duration_Minutes': Calculate from other variables using the formula:

    (Trip_Price - Base_Fare - Per_Km_Rate * Trip_Distance_km) / Per_Minute_Rate

  • 'Trip_Distance_km': Calculate from other variables using the formula:

    (Trip_Price - Base_Fare - Per_Minute_Rate * Trip_Duration_Minutes) / Per_Km_Rate

  • 'Trip_Price': Calculate using the formula:

    Per_Minute_Rate * Trip_Duration_Minutes + Per_Km_Rate * Trip_Distance_km + Base_Fare

# Using KNN to impute missing categorical values
for cat in categorical_columns:
    # Selecting numeric variables and one categorical variable at a time
    select_col = quantitative_columns.copy()
    select_col.append(cat)
    
    df_knn = df[select_col]
    
    # Separating data into training (rows without NaN) and test (rows with NaN)
    # The ~ operator inverts boolean values, selecting rows without NaN
    x_train = df_knn[~df_knn.isnull().any(axis=1)]
    x_train = x_train.drop(cat, axis=1)
    x_test = df_knn[df_knn.isnull().any(axis=1)]
    x_test = x_test.drop(cat, axis=1)

    y_train = df_knn[~df_knn.isnull().any(axis=1)]
    y_train = y_train[cat]
    y_test = df_knn[df_knn.isnull().any(axis=1)]
    y_test = y_test[cat]
    
    # Normalizing data with StandardScaler
    scaler = StandardScaler()
    x_train = scaler.fit_transform(x_train)
    x_test = scaler.transform(x_test)

    # Creating and training KNN model with 5 neighbors
    knn = KNeighborsClassifier(n_neighbors=5)
    knn.fit(x_train, y_train)

    # Making predictions and filling missing values
    y_pred = knn.predict(x_test)
    
    for i, pred in zip(y_test.index, y_pred):
        df.loc[i, cat] = pred
