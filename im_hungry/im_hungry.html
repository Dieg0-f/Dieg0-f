- AI’m Hungry –

AI’m Hungry is a multilingual AI-powered chatbot that helps University of Strasbourg students access daily Crous restaurant menus with allergen alerts and meal-time relevance through an intuitive, interactive interface.

Project Overview

"AI’m Hungry" is an interactive chatbot application designed to assist users in discovering daily menus at the Crous restaurants of the University of Strasbourg. This project leverages web scraping, natural language processing, and machine learning to provide relevant menu information, including allergens and meal schedules, through an intuitive interface. The application supports multilingual queries and offers an enriched user experience via Dash, Streamlit, or WhatsApp.

Project Objectives

  • Menu Retrieval: Scrape and store menu information from Crous restaurant websites
  • Interactive Chatbot: Enable users to query menus using natural language
  • Advanced Features: Include allergen alerts, meal timing relevance (lunch/dinner), and multilingual support

File Structure and Functionality

requirements.txt: Lists the required Python libraries and dependencies:

  • NLP frameworks: langchain, transformers, sentence-transformers
  • Web scraping: beautifulsoup4, requests
  • Other utilities: pandas, streamlit, numpy, google-cloud-translate

menu_functions.py: Core logic for menu retrieval and processing:

  • HF_model(): Uses Hugging Face models for named entity recognition (NER)
  • get_link(): Extracts restaurant-related links based on user queries
  • get_menu(): Retrieves and cleans menus for specific restaurants
  • add_allergens(): Identifies allergens in menu items
  • translate_text() and detect_language(): Handle multilingual support

menu.py: Sets up the Streamlit UI with chatbot interface and session state

Application.ipynb: Prototyping notebook for chatbot features and API integration

Giallozafferano.ipynb: Experimental notebook for recipes or feature exploration

scrap_new_menu.py: Script for full-day menu scraping and ingredient analysis

Key Features

  • Web Scraping: Retrieves and cleans menu data using BeautifulSoup
  • AI-Powered Chatbot: Uses LangChain for context-aware responses
  • Allergen Detection: Alerts users based on an allergen database
  • Multilingual Support: Detects and translates language with Hugging Face and Google Translate
  • User-Friendly Interface: Streamlit-based interface with quick question options

How to Run

  1. Install dependencies: pip install -r requirements.txt
  2. Set up environment variables: Add API keys to a .env file
  3. Launch the app: streamlit run menu.py
  4. Interact with the chatbot: Ask questions or use pre-suggested queries
